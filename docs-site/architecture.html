Architecture

Confidence formula

The attack-state engine scores progression deterministically:

confidence = (confirmed × 0.50) + (supporting × 0.20) + (progression × 0.30)
escalate         when confidence ≥ 0.70
escalate (page)  when confidence ≥ 0.85
